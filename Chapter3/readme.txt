gCRM palaeointensity method which requires the user input: 
current variables set for greigite in this version (Ms)

User runs each of the segments in order and is asked what to do at each step. 

Input files requires are the .frc file, .sirm file, .nrm files in the same folder (e.g., name.frc, name.sirm, name.nrm)
with the .nrm and .sirm having the format the same to to the example files included here. 

For the Maurel code, the selected fields are defined in the functions file.
